# 手势控制 Flappy Bird 游戏说明书

## 项目简介
这是一个基于 Web 的 Flappy Bird 游戏，利用 Google MediaPipe Hands 技术实现手势控制。玩家可以通过摄像头捕捉手势来控制小鸟的飞行，无需键盘或鼠标。

## 运行方法
由于浏览器安全限制，无法直接打开 HTML 文件访问摄像头，需要通过本地服务器运行。

1.  **启动本地服务器**：
    *   如果你安装了 Python，请在项目目录下运行：
        ```bash
        python3 -m http.server 8000
        ```
    *   或者使用 VS Code 的 "Live Server" 插件。
2.  **打开游戏**：
    *   在浏览器中访问 `http://localhost:8000`（或相应的端口）。
3.  **允许摄像头权限**：
    *   浏览器提示时，请点击“允许”以授权访问摄像头。

## 游戏玩法

### 1. 准备阶段
*   观察摄像头画面下方的状态文字，等待显示 **"Camera Active"**。
*   此时你应该能在画面中看到绿色的手部骨架追踪线。

### 2. 开始游戏
*   **捏合手势启动**：对着摄像头做一个“捏合”动作（拇指和食指捏在一起），或者点击屏幕上的“Start Game”按钮。
*   游戏会有 **3秒倒计时**，请做好准备！

### 3. 控制方式
*   **跳跃 (Jump)**：
    *   **动作**：**捏合**拇指和食指（就像捏住一样东西）。
    *   **效果**：小鸟会向上飞一下。
    *   **提示**：画面上会出现黄色圆圈提示捏合成功。需要松开再捏合才能再次跳跃。
*   **悬停 (Hover) - 防疲劳模式**：
    *   **动作**：**食指向上** 👆（食指伸直向上，其他四指握拳）。
    *   **效果**：小鸟会**停止下坠**，保持在当前高度悬浮。
    *   **用途**：当你觉得累了，或者想精准控制高度通过管道时，可以使用此手势。画面上会出现青色圆圈提示悬停模式。

## 常见问题 (Troubleshooting)
*   **摄像头没反应？**
    *   请检查是否有其他软件占用了摄像头。
    *   确认浏览器地址栏是否允许了摄像头权限。
*   **手势识别不灵敏？**
    *   **光线**：请确保环境光线充足，不要背光。
    *   **距离**：手部距离摄像头保持适中（约 0.5 - 1 米），确保整个手掌都在画面内。
    *   **背景**：背景尽量简洁，避免与肤色相近的颜色。
*   **游戏卡顿？**
    *   关闭其他占用 CPU/GPU 的网页或程序。
    *   MediaPipe 需要一定的计算资源，建议在较新的电脑上运行。
